## 📈 Reportes IA — 4 Tipos de Análisis

### 1️⃣ Resumen Ejecutivo
📋 Visión general para directorio  
✅ Estado actual + Logros + Desafíos + Recomendaciones

### 2️⃣ Cumplimiento Normativo
✓ Análisis detallado de cumplimiento  
✅ Brechas identificadas + Riesgos críticos + Plan de mejora

### 3️⃣ Análisis de Riesgos
⚠️ Identificación de riesgos operacionales  
✅ Riesgos críticos + Matriz de riesgo + Mitigación

### 4️⃣ Alertas Críticas
🚨 Lista priorizada por urgencia  
✅ Críticas (24h) + Altas (3 días) + Medias (1 sem) + Bajas

**Cómo usarlos:**
1. Ve a "Reportes IA"
2. Selecciona el tipo que necesitas
3. Haz clic en "Generar Análisis"
4. En 2-3 minutos: Análisis completo impulsado por OpenAI GPT-4

## 🎯 Casos de Uso Reales

### 📍 Lunes por la Mañana — Gerente General
1. Abre **Centro de Operaciones** → Revisa métricas
2. Abre **Reportes IA** → "Resumen Ejecutivo"
3. Pregunta al **Asistente IA**: "¿Hay problemas críticos hoy?"
4. Lee recomendaciones de IA → Toma decisiones

⏱️ Tiempo total: 5 minutos

### 📍 Martes — Supervisor de Mantenimiento
1. Abre **Mantención** → Ve equipos con mantenimiento atrasado
2. Abre **Bodega** → Verifica disponibilidad de repuestos
3. Abre **Compras** → Crea órdenes si falta stock
4. Consulta **Alertas** → Ve qué se marcó como "en progreso"

⏱️ Tiempo total: 15 minutos

### 📍 Viernes — Reporte Ejecutivo
1. Abre **Reportes IA**
2. Selecciona "Resumen Ejecutivo"
3. Genera análisis (2-3 min)
4. Copia y comparte con directorio

✅ Reporte listo para junta
